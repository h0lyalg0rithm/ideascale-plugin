== Description ==

Display the top ideas from your ideascale community as a widget on your website.

== Note == 

Requires API KEY from ideascale community.

== Installation ==

1. Upload `wp-ideascale.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
